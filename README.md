# WDI – Wstęp do Informatyki

Repozytorium zawiera rozwiązania prostych zadań programistycznych realizowanych w ramach przedmiotu **Wstęp do Informatyki (WDI)** na **Akademii Górniczo-Hutniczej w Krakowie, Wydziale Informatyki**.

---

## Opis

Zadania obejmują podstawowe elementy programowania i algorytmiki, takie jak:
- instrukcje warunkowe i pętle,  
- operacje na tablicach i ciągach liczb,  
- prosta rekurencja,  
- podstawowe struktury danych i operacje na nich.  

Celem kursu było wprowadzenie do myślenia algorytmicznego oraz nauka pisania poprawnych i czytelnych programów.

---

## Technologie

- **Język programowania:** Python  
- **Środowisko:** PyCharm  
- **Narzędzia:** Git, GitHub  

---

## Czego się nauczyłem

Podczas zajęć zdobyłem podstawowe umiejętności w:
- pisaniu prostych programów w języku Python,  
- analizie i rozwiązywaniu problemów algorytmicznych,  
- korzystaniu z repozytorium Git i pracy z kodem w IDE,  
- tworzeniu czytelnego, modularnego kodu.
